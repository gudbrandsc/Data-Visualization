California Counties Choropleth Map

This choropleth map displays a geographic representation of the different counties in California. The color scale is based on the violent crime rate for the selected time, where the min, mean and max are displayed in the legend. It's important to note that the color scale will change based on the selected year so the same color might not represent the same value in a different year. This visualization comes with multiple interactive features, and can also be used to select the county displayed in the multi-line chart.
